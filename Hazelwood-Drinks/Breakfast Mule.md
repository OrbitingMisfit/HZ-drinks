# Ingredients
- 1.5 Oz [[Ketle One Peach]]
- 0.5 Oz Orange Juice
- 0.5 Oz [[Agave Simple Syrup]]
- 3.0 Oz Ginger Beer
- Lime Garnish

# Instructions
1. Build in glass
2. Add Kettle One Peach
3. Add [[Agave Simple Syrup]]
4. Add Orange Juice
5. Add GingerBeer
6. Finish with ice
7. Garnish with Lime wedge
