# Ingredients
- 2.0 Oz [[Pepper Vodka (Batch)]]
- 5.0 Oz [[Bloody Mary Mix (Batch)]]
- 1x [[Bloody Mary Skewer]]



# Instructions
1. Build in [[Bloody Rimm]] Pint Glass
2. Add ice
3. Add Vodka
4. Top with [[Bloody Mary Mix (Batch)]]
5. Garnish with Skewer and straw
